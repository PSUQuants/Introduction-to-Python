Here is the formula for a standard Capital Asset Pricing Model, that we have visualzied in Python:
Ra = Rrf + [Ba x (Rm – Rrf)], where

Ra = Expected return on a security
Rrf = Risk-free rate
Ba = Beta of the security
Rm = Expected return on market

Note: Risk Premium = (Rm – Rrf)
